This page provides information for how to troubleshoot common problems and errors.


== Starting with Status Code "000" on Imports
.Issue
* The issue occurs when the script xref:../scripts/import.adoc[`./import-onecx.sh`] is executed (importing OneCX data)

.Status code 000 on initial imports
image::status-code-000-import-onecx.sh.png[]

.Solution
* Check if the hosts files is up-to-date. See xref:../setup.adoc#hosts[Setup hosts] instructions.


== Cannot successful pull the latest version onecx-local-env
.Occurrence
* On `git pull` (get latest of onecx-local-env)

.Untracked local files on pulling
image::untracked-files-on-pull.sh.png[]

.Probably also your init_data/traefik/active was created by root
image::traefik-active_created-by-root.sh.png[]

.Solution
* Remove the active directory of Traefik
** `sudo rm -rf init_data/traefik/active`
* Reset local repo
** `git reset --hard origin/main`
